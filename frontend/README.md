# Trades Frontend (React + Material UI)

## Run
1. Install dependencies:
   npm install
2. Start dev server on localhost:4000:
   npm run dev

## Notes
- API base URL: http://localhost:8080
- Auth uses HttpOnly cookie (`TRADES_AUTH`) with `withCredentials=true`.
- Ensure backend CORS allows http://localhost:4000.

## Main Screens
- /login
- /users
- /vendors
- /trades
- /trades/:id
